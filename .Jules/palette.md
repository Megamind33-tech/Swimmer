## 2026-05-03 - Accessibility Label Gap Analysis
**Learning:** Discovered that key navigation components lacked foundational accessibility attributes (`aria-label`, `title`), despite being icon-driven. This suggests a pattern where visual design was prioritized over accessible markup for structural UI elements.
**Action:** Implemented labels on the primary navigation. Moving forward, will proactively audit all icon-only interactive elements across overlays and HUDs as a baseline standard for UI enhancements.
